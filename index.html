  • Langkah ke 2 itu copy yg ada tulisan remotenya dari github
  • Langkah ke 3, kita bisa ketik git add .
  • Langkah ke 4, kita bisa masukin pesan commit pake cara git commit -m "pesan komitnya"

  • Cara bikin branch ketik git branch -M "nama branch", biasanya nama branch utama itu main
  • Langkah 6, setelah itu tinggal push ketiknya git push -u origin namabranch
